## A coding workspace based on [Alacritty](https://alacritty.org/), [Zellij](https://zellij.dev/) and [LazyVim](https://www.lazyvim.org/)
### How to use ?
Execute every command in [init.sh](https://github.com/xusd320/xsh/blob/master/init.sh) line by line.
Then open Alacritty, type `zellij`, open nvim. Then coding out of box on MacOS.
